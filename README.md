# Steam Anchor

Easily downgrade your Steam client and prevent it from automatically updating on Windows.

## Features

- **Live Version List**: Fetches the latest available Steam versions online.
- **One-Click Downgrade**: Choose a Steam version from the list and apply.
- **Update Blocking**: Blocks Steam from updating itself on startup. You can unblock it anytime.

## Download

Grab the `.exe` from the [**Releases**](https://github.com/pintoso/SteamAnchor/releases/latest) tab.

## Running from Source

1. Clone this repository.
2. Install Python.
3. Run `python main.py` or build your own executable with `build.bat`.
